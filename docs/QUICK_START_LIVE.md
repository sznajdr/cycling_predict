# Live Race Dashboard - Quick Start

## 🚀 Launch Now (Before Race Starts!)

### Step 1: Open Terminal

In VS Code PowerShell:
```powershell
cd path\to\cycling_predict
```

### Step 2: Launch Dashboard

```powershell
streamlit run scripts/live_race_dashboard.py
```

This will:
1. Start the Streamlit server
2. Open your browser automatically
3. Show the race picker
4. Allow live monitoring

### Step 3: Use the Dashboard

**Race Selection:**
- Pick "Paris-Nice 2026" from sidebar
- Select "Stage 1 - flat (170.9km)"

**Tabs:**
1. **📊 Live Race** - Shows live results when race starts
2. **💰 Value Bets** - Model predictions (run `rank_stage.py` first!)
3. **👥 Startlist** - All riders with specialty scores
4. **⚡ Alerts** - Attack detection (Strategy 12)

### Step 4: Prepare Value Bets (Before Race!)

In another terminal:
```powershell
python rank_stage.py paris-nice 2026 1 --run-models --save
```

Then refresh the dashboard!

## 📊 Data Sources

1. **Race/Startlist** - Your local SQLite database (cycling.db)
2. **Live Results** - Scraped from PCS race page every 30s
3. **Predictions** - Generated by your models (Strategy 1, 2)
4. **Odds** - Need to run fetch_odds.py for real odds

## ❓ Troubleshooting

### "Streamlit not found"
```powershell
pip install streamlit pandas beautifulsoup4 requests
```

### "Database not found"
```powershell
python -m pipeline.runner
```

### "No model predictions"
```powershell
python rank_stage.py paris-nice 2026 1 --run-models --save
```

### Page won't load
- Check firewall isn't blocking port 8501
- Try: `streamlit run scripts/live_race_dashboard.py --server.port 8502`
